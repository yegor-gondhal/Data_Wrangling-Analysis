PURPOSE:
The Purpose of this project is to learn the process of Data Wrangling in order to correctly and efficiently
clean data so that it is ready to use. Additionally, I did some Data Analysis and learned about the 
Autocorrelation Function, which will be helpful in the future when I will need to find patterns in data.

Files:
dirty_cafe_sales_csv : A kaggle practice dataset that includes coffee shop data that needs to be cleaned up

clean_data.csv : My cleaned version of the data which I obtained by running clean.py
                 Includes empty cells, which maximizes the number of rows incase not all columns need to be used

clean.py : A program that cleans the data and provides a function for extracting certain columns with as many rows as possible

analysis1.py : A program that analyzes year-long smoothie and coffee buying trends. Uses ACF for analysis as well.

analysis2.py : A program that analyzes correlation between certain products and the quantity of them purchased.
